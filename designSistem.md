# Para select, sempre use o componente src/components/ui/Select.css
Esse componente Seletor Suspenso Personalizado" que suporta ícones, pesquisa (se adicionada) e renderização de opções personalizadas, diferente do select nativo do navegador usando um elemento Trigger (a caixa em que você clica) e um elemento Dropdown (a lista que aparece).

# Para botão, sempre use o componente src/components/ui/Button.css
botões do estilo /src/styles/buttons.css

# Para modal
Para modal, o o seguinte padrão:
Modal com Header, body e footer. Botões de ação sempre no footer. O botão de fechar sempre no canto superior direito. O modal deve ter um overlay escuro.

# Para alertas de confirmação
Alertas de confirmação sempre use modal com icones Lucide e botões do estilo /src/styles/buttons.css

# Para alerta pós confirmação
Para alertas pós confirmação use sempre AlerttToast que já temos no projeto
